# 🚖 RideEase – Smart Cab Booking System

RideEase is a Java-based smart cab booking application designed to provide a seamless, secure, and user-friendly ride booking experience. Built using Java and MySQL, the system enables users to book rides, manage bookings, and access essential travel features through an intuitive interface.

Unlike a traditional cab booking application, RideEase incorporates innovative features such as **Emergency SOS**, **Ride Scheduling**, **Eco Ride** and **Favorite Locations**, making it a modern and practical transportation solution.

---

## ✨ Features

### 👤 User Management
- User Registration & Login
- Secure User Authentication
- Profile Management

### 🚖 Ride Booking
- Book rides instantly
- View driver details before confirming the ride
- Cancel bookings
- View complete ride history

### 📅 Ride Scheduling
- Schedule rides for a future date and time
- Manage upcoming scheduled rides
- Receive booking confirmation for scheduled trips

### 🌿 Eco Ride
- Choose between Standard Cab and Electric Cab
- View estimated carbon emissions saved by selecting an Electric Cab
- Encourage environmentally friendly transportation

### 🛡️ Emergency SOS
- Dedicated SOS button during active rides
- Simulates notifying the user's emergency contact
- Displays driver information, vehicle details, and nearest police station for emergency assistance

### 📍 Favorite Locations
- Save frequently visited places such as:
  - 🏠 Home
  - 🏢 Office
  - 🎓 College
- Book rides quickly without repeatedly entering addresses

### 👨‍💼 Admin Module
- Manage users and drivers
- Monitor ride bookings
- View ride statistics
- Manage scheduled rides
